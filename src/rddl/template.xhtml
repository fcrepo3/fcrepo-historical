Introduction

This document describes the http://www.fedora.info/definitions/1/0/api/ namespace. It also contains a directory of links to related resources, using Resource Directory Description Language.

Related Resources

Schemas for XML Schema

DTD

A DTD XMLSchema.dtd for XML Schema. It incorporates an auxiliary DTD, datatypes.dtd.

XML Schema

An XML Schema schema document for XML Schema schema documents. Note the simple type formerly called 'decimal', which was briefly renamed 'number', is back to being called 'decimal'. Schema documents validated with this schema document between 16 March and 30 March 2001 will need to be modified if they used the 'number' type.

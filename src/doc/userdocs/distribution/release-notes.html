Fedora 2.2.1 Release Notes

August 13th, 2007

Table of Contents

  1. Bug Fixes
  2. Known Issues
  3. Installation
  4. Historic Release Notes (prior releases)

Fedora 2.2.1 is a minor maintenance release that addresses several installation and security-related bugs discovered with Fedora 2.2.

  1. Bug Fixes

    The following bugs have been addressed for this release.

    • Bug #1661682: Bundled Postgres fails to copy during install
    • Bug #1661683: Postgres NumberFormatException at startup
    • Bug #1686346: WSDL provided under target /fedora/wsdl is not public
    • Bug #1721620: IllegalStateException when authorization header not provided
    • Bug #1731608: LDAP, JNDI, and Servlet Filter Cache issues
    • Bug #1768738: Saxon Servlet not respecting base URLs
    • Bug #1772984: Automatic rename of do table to dobj sometimes fails

  2. Known Issues

    Please see the Fedora Commons Bug Tracker for an up-to-date list of outstanding bugs.

  3. Installation

    For instructions on installing and upgrading Fedora, please consult the Installation Guide and the Fedora Upgrade and Migration Guide.

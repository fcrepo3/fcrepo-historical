2. Server

server/bin
Contains server command-line utilities, including fedora-rebuild, fedora-reload-policies, and validate-policy.
server/config
Contains the server configuration files, including fedora.fcfg and log4j.properties
server/logs
Contains the server log files.
server/access
Stylesheets for the default disseminator. All objects in the repository automatically subscribe to this disseminator, which provides methods for listing and retrieving the components of the object.
server/xsd
W3 XML schemas, for reference and validation support.
server/schematron
Files supporting the validation of serialized digital objects.

If the included Tomcat servlet container was chosen during the installation (i.e. either a "quick" install or "included" servlet container was selected as part of a custom install), tomcat will contain an installation of Tomcat 5.0.28.

If the included McKoi database was chosen during the installation (i.e. either a "quick" install or "included" was selected as part of a custom install, mckoi1.0.3 will contain the configuration and data for the embedded McKoi instance.

Fedora Framework Services

  1. Directory Ingest Service (DirIngest)  - ingest directories of files into Fedora
  2. OAI Provider Service (PROAI) - a configurable, flexible OAI Provider service
  3. Fedora Generic Search Service (GSearch) - index datastreams and disseminations with either a Lucene or Zebra backend
  4. Fedora Journaling Service - create a backup or mirror repository

2  Fedora Repository Server (with Core Repository Web Service Interfaces)

 

As of Fedora 2.1b, the Fedora Service Framework is being introduced to facilitate the development of new services to be used in conjunction with the Fedora repository service.  The Fedora repository service, itself, was developed to fit into a broader service-based architecture (SOA), and it provides web services interfaces to its core functionality.  The Fedora repository services is depicted below.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

2   The Fedora Service Framework

3 Future Services in the Fedora Service Framework (2005-2007)

 

The Fedora Service Framework will evolve with new services over the timeframe of the current Fedora Phase II project (2005-2007).   Depicted below are services that are envisioned.

  • Preservation Integrity Service
  • Workflow Service
  • Search Service
  • PID Resolution Service (for Federations)
  • Preservation Monitoring Service
  • Event Notification Service (Publish/Subscribe)
  • Open URL Access Point

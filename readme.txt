Fedora Release 1.1.1 - September 5, 2003
---------------------------------------------------------------
This is a full source code release of Fedora,
downloaded from http://www.fedora.info/release/1.1.1/

Installation instructions and release notes can be found at:
    http://www.fedora.info/release/1.1.1/userdocs/

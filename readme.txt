Fedora Release 1.0 - May 16, 2003
----------------------------------
This is a full source code release of Fedora,
downloaded from http://www.fedora.info/release/1.0/

Installation instructions and release notes can be found at:
    http://www.fedora.info/release/1.0/userdocs/

When the server is installed, the above documents,
along with others, are available locally at
http://hostname:port/userdocs/
